## 🎯 الغرض من المنصة

منصة **مسار** هي أداة رسمية مخصصة لـ:

- إدارة قاعدة البيانات الوطنية لرؤساء أقسام التربية الولائيين ونوابهم
- متابعة برامج التكوين والتدريب
- توثيق المسار المهني للإطارات التربوية
- تسهيل التواصل والتنسيق بين لجان التكوين الولائية
